# מערכת תיאום אימונים משותפים

מערכת לתיאום אימונים משותפים בין טייסות מסוקים לכוחות קרקעיים.

בנוי עם Next.js 14 (App Router) + Vercel KV לאחסון משותף.

## הרצה מקומית (לא חובה — אפשר לפרוס ישירות ב-Vercel)

```bash
npm install
npm run dev
```

## פריסה

ראו את ההוראות שקיבלתם בצ'אט להקמת GitHub + Vercel + Vercel KV.
